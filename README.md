## Usage
 
### How to Include
- Place `lib/` and `expander.py` in the same directory as your source file.
- Include headers like `#include "lib/utils/fast_io.hpp"` in your source file.
- Add `-I.` to your compiler flags.
 
### Environment
- C++20 or later is required. We recommend compiling with `-std=c++23 -O2`.
- Some files depend on GNU extensions.
- Performance is generally better on Unix-like systems.

### Library Code
- We recommend using the provided `template.cpp`. Otherwise, you will need to include `<bits/stdc++.h>`.
- Always use half-open intervals `[left, right)`. All data structures are 0-indexed.
- We assume `uint32_t` is large enough to store sizes. Note that `size = 0` may lead to unexpected behavior.
- Do not call low-level functions or change their arguments unless you clearly understand what you are doing.
- This library performs only limited safety checks. Please ensure that all input arguments are valid.

### Expander
- `expander.py` was generated by ChatGPT. It is used to expand the code into a single file, similar to the expander in ACL.
- You can simply run `python3 expander.py ${file} -o submit.cpp` to use the expander. For full usage instructions, see the source file.
